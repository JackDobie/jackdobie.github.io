Robo Holdout

This is a first person shooter game that I made for Global Game Jam 2020. It was made over the course of 48 hours, starting on the 31st January 2020. This was my first game jam, and it was a lot of fun. I plan on doing it again next year.
Link to the Game Jam page, with a link to download: https://globalgamejam.org/2020/games/robo-holdout-1
